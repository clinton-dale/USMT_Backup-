# USMT_Backup-
Modified USMT Backup with xml and scripts
This is the USMT version 10 modified with config and specific exclude xml files. 

This program has been modified to only backup user specific information, not general windows core or other functions as a result over 95% of the config file has been set to migrate=no.  Before using the included config files please ensure that the sections you are looking to migrate have not been turned off.
